cli-template Structure

cli-template is example template project for general application. You can fork it to accelerate yor application dev process.

  • 1. How to run : go run .
  • 2. How to gen-api : go run . gen_api
  • 3. Two types of config mod with config-file in configs folder
  • 4. API-Docs /swagger/index.html

  • 5. Log system with file storage features
    • 5.1 to show all history logs : go run ./ log
    • 5.2 to show only [error,panic,fatal] history logs : go run ./ log --only_err=true

  • 6. Install your service on your OS
    • 6.1 install service : sudo go run ./ service install
    • 6.2 start service : sudo go run ./ service start
    • 6.2 restart service : sudo go run ./ service restart
    • 6.3 stop service : sudo go run ./ service stop
    • 6.4 remove service : sudo go run ./ service remove

  • 7. Current supported plugins
    • 7.1 SqlData database [Gorm based]
    • 7.2 Redis
    • 7.3 HttpServer [Echo based]
    • 7.4 Reference [A local memory key-value reference system]
    • 7.5 ElasticSearch
    • 7.6 Single Process [a job only run in one process among all auto-scaling instances environment]
